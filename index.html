About

Assistant Professor of Computing Science, resarching on Networked Computer Systems and Cyber Security. I study Operating Systems and their Networking stack. Currently, I'm conducting projects about the 5th generation of mobile networks (5G & B5G) and applications like Intelligent Transportation Systems and (Industrial) Internet of Things.

My research interests include, among others: Mobile (Multi-access) Edge Computing (MEC), Internet of Things (NB-IoT), SDR/BLE/Wi-Fi, LoRa/LoRaWAN, Cyber Security, Linux Kernel, Operating Systems Development, netdev, eBPF, reverse engineering, pwning, ROP, and machine learning.

AB-TRAP

The increase of connected devices and the constantly evolving methods and techniques by attackers pose a challenge for network intrusion detection systems from conception to operation. As a result, we see a constant adoption of machine learning algorithms for network intrusion detection systems. However, the dataset used by these studies has become obsolete regarding both background and attack traffic. This work describes the AB-TRAP framework that enables the use of updated network traffic and considers operational concerns to enable the complete deployment of the solution. AB-TRAP is a five-step framework consisting of ( i ) the generation of the attack dataset, ( ii ) the bonafide dataset, ( iii ) training of machine learning models, ( iv ) realization (implementation) of the models, and ( v ) the performance evaluation of the realized model after deployment. We exercised the AB-TRAP for local (LAN) and global (internet) environments to detect TCP port scanning attacks. The LAN study case presented an f1-score of 0.96, and an area under the ROC curve of 0.99 using a decision tree with minimal CPU and RAM usage on kernel-space. For the internet case with eight machine learning algorithms with an average f1-score of 0.95, an average area under the ROC curve of 0.98, and an average overhead of 1.4% CPU and 3.6% RAM on user-space in a single-board computer. This framework has the following paramount characteristics: it is reproducible, uses the most up-to-date network traffic, attacks, and addresses the concerns to the model’s realization and deployment.

SCREEN

Este projeto automatiza a obtenção de firmwares de roteadores domésticos. Com a grande adoção de home-office em um mundo pós-COVID, esses ativos tornam-se alvos fáceis para atacantes e com isso ganham acesso a redes de empresas. Esperamos obter 30 mil firmwares (já temos 5 mil) e queremos minerá-los para obter falhas críticas. Aprendizado de máquina será essencial para obter os artefatos mais comuns e direcionar o trabalho.

DETECTION

Fake news é um grande problema de nossa sociedade atual. Ferramentas para entender a disseminação de informação por agente maliciosos ainda são incipientes. Neste projeto utilizaremos sondas para redes sociais e com isso entenderemos a estrutura de organizações para disseminação de informação.

Controller

Impulsionado pela adoção do 5G novas aplicações começam a surgir. Elas representam grandes avanços tecnológicos, dentre eles os veículos autônomos se destacam. No entanto, a exploração de vulnerabilidades neste contexto podem ser catastróficas, inclusive possibilitando o foco em pessoas alvo. Neste trabalho exploraremos como funciona a implementação deste tipo de aplicação em testbed real e como é caracterizado o tráfego típico desta aplicação. Algoritmos de aprendizado de máquina são utilizados para implementar mecanismos de detecção de anomalias para proteção contra ataques cibernéticos.

Ransomware

Ransomwares são malwares com potencial catastrófico em infraestrutura de sistemas computacionais e possuem a capacidade de deixar inoperantes serviços críticos para as organizações. Entender o perfil desses softwares em execução são essenciais para implementação de mecanismos de detecção inteligente e práticas forenses mais efetivas. Neste projeto utilizamos técnicas de inteligência artificial para maximizar a qualidade das informações coletadas em logs.
